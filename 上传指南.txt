# 上传到Gitee的简便方法

## 方法：在线复制粘贴

1. 打开Gitee仓库的在线编辑页面：
```
https://gitee.com/Mason3210/subway-dashboard/src/branch/master
```

2. 点击右上角 **"在线编辑"** 按钮（蓝色按钮）

3. 会打开一个编辑器页面

4. **删除里面的所有内容**，然后**粘贴我给你的index.html全部代码**

5. 填写提交信息：`添加看板页面`

6. 点击 **"提交"**

---

## 如果在线编辑找不到，试试这个：

直接访问这个地址，点击"上传文件"按钮：
```
https://gitee.com/Mason3210/subway-dashboard/upload
```
