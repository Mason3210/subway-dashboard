# 📸 Gitee 上传图片指南（图文）

## 步骤1: 登录后进入仓库

登录后访问: https://gitee.com/projects/new#git-template

点击 **"导入已有仓库"** 或 **"创建空白项目"**

---

## 步骤2: 创建仓库

```
仓库名称: subway-dashboard
路径:     subway-dashboard  
公开度:   公开 ✅ (必须选公开)
语言:     HTML
```

点击 **"创建"**

---

## 步骤3: 上传文件（关键！）

创建后进入仓库页面，你会看到：

```
┌─────────────────────────────────────────┐
│  subway-dashboard                       │
│  ─────────────                          │
│  📁 代码   📋 讨论   ⚙️ 服务   📖 Wiki  │
│                                          │
│  ┌────────────────────────────────┐   │
│  │ 📁 subway-dashboard            │   │
│  │ README.md                      │   │
│  │                                │   │
│  │ + 上传                         │   ← 点击这个！
│  │ 📥 下载                        │   │
│  └────────────────────────────────┘   │
│                                          │
│  ┌────────────────────────────────┐   │
│  │ 拖拽文件到此处或点击上传        │   │
│  └────────────────────────────────┘   │
└─────────────────────────────────────────┘
```

**找到橙色的 "+ 上传" 按钮，点击它！**

---

## 步骤4: 具体操作

1. 点击 "上传" 后，会出现选择文件框
2. 把 `C:\subway-safety-dashboard\index.html` 选上
3. 填写下方 "提交信息": `Initial upload`
4. 点击绿色的 "提交" 按钮

---

## 步骤5: 启动 Pages

1. 提交成功后，在仓库页面找 **"服务"** 菜单
2. 点击 **"Gitee Pages"**
3. 点击 **"启动"** 按钮
4. 等待几秒，会显示你的网址
