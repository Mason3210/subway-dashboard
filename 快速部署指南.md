# 🚀 5分钟快速部署指南

## 步骤 1: 注册/登录 Gitee

打开 https://gitee.com 并登录

## 步骤 2: 创建仓库

1. 点击右上角 "+" → 新建仓库
2. 填写:
   - 仓库名称: `subway-dashboard`
   - 路径: `subway-dashboard`
   - 选择: 公开
3. 点击"创建"

## 步骤 3: 上传 index.html

1. 进入仓库后，点击"上传"
2. 把 `C:\subway-safety-dashboard\index.html` 拖进去
3. 填写提交信息: "Initial upload"
4. 点击"提交"

## 步骤 4: 启动 Gitee Pages

1. 仓库页面 → 点击"服务"
2. 点击"Gitee Pages"
3. 点击"启动"
4. 等待几秒，显示网址

## 步骤 5: 访问

点击生成的网址即可访问看板！
